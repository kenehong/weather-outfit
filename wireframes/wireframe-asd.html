ASD Design Highlights

  • Predictability: Layout ratio (70/30) and element order NEVER changes. Only info amount changes.
  • Sensory Profiles: Standard (full animation) → Calm (no particles, -30% saturation, 2x slower transitions) → Still (no animation, flat color).
  • 3 Learning Levels: L1 = emoji + 1 word (no numbers). L2 = temp + sentence + hourly icons. L3 = full data + search.
  • Visual Schedule: L1 only. Weather → Dress checklist → Ready to go! Tap each step. No auto-advance.
  • Transition notice: "Weather is changing..." shown 2s before any scene change.
  • Interactions: Single tap only. No swipe, long-press, or double-tap. Touch targets 56px+.
  • Parent settings: Hold ⚙️ 3 seconds. Controls sensory profile, level (lockable), schedule on/off.
  • No infantilization: Same character for all ages. Only info complexity changes.
