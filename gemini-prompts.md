# Gemini Image Prompts — Weather Outfit App

> Shiba Run 픽셀아트 스타일로 통일된 캐릭터 + 배경 에셋
> 캐릭터마다 2-3 프레임 (idle 루프 애니메이션용)
> Smart display (1024×600 landscape)

---

## 🎨 공통 스타일 가이드

기존 Shiba Run 캐릭터 스타일 레퍼런스:
- **Pixel art** style, chunky pixels, clean black outline
- Kawaii/cute proportions (big head, small body, big eyes)
- Orange/warm color palette with **pink cheeks**
- White belly/chest area
- Transparent background, PNG
- Simple but expressive

> 모든 프롬프트 앞에 이 스타일 블록을 붙여서 일관성 유지:

```
Style: Cute pixel art character, chunky pixels with clean black outlines,
kawaii proportions (big head, small body, large expressive eyes),
pink rosy cheeks, warm colors, simple and readable at small sizes.
Transparent background, PNG. Match the style of the reference shiba inu
pixel art character exactly.
```

## 🎬 애니메이션 구현

앱에서 CSS animation으로 프레임 순환:

```css
.character {
  animation: idle-loop 1.5s steps(1) infinite;
}

/* 3프레임 루프: frame1(0.5s) → frame2(0.5s) → frame3(0.5s) */
@keyframes idle-loop {
  0%    { background-image: url('duck-1.png'); }
  33.3% { background-image: url('duck-2.png'); }
  66.6% { background-image: url('duck-3.png'); }
}
```

Calm 모드: `animation-duration: 3s` (2배 느리게)
Still 모드: `animation: none` (첫 프레임 고정)

## 💡 Gemini 사용 팁

1. **레퍼런스 이미지 첨부**: idle.png를 Gemini에 업로드하고 "match this exact pixel art style" 지시
2. **한 캐릭터 3프레임 연속 생성**: 같은 채팅에서 Frame 1 → 2 → 3 순서로 요청하면 일관성↑
3. **크기**: 512×512로 생성 → 앱에서 200px로 리사이즈 (픽셀아트는 nearest-neighbor 스케일링)
4. **배경도 같은 세션에서**: 캐릭터 생성 직후 배경 생성하면 색감 매칭↑
5. **Calm 모드 대비**: 채도 -30% 적용되므로 원본 채도 적당히 유지
