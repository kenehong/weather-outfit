Design Notes

  • Layout: Scene (left ~70%) + Info Panel (right ~30%). Landscape optimized for smart displays.
  • Character: Centered in the scene. Gemini-generated images (PNG, transparent bg) will replace the placeholder.
  • Info Panel: Always visible on the right. Shows advice text, temperature, condition, and search.
  • Scenes needed: At minimum 4 (sunny, cloudy, rainy, snowy). Each with day/night variant = 8 CSS themes.
  • Character images needed: 6 outfits × weather overlay (umbrella/scarf) = ~8-10 images from Gemini.
  • Smart Display deployment: Can run as a fullscreen web app via the device's built-in browser (Silk on Echo, Chrome on Nest Hub via kiosk mode).
